# Gomoku Mobile Game (Android) 五子连珠小游戏 (Android)
This is my first self-taught Android project. 这是我的第一个自学 Android 作品。

## Features 功能
-4 difficulty levels with different colors 4 色难度挑战
-High score record tracking 历史最高分记录
-Customizable background color 自定义背景颜色
-Smooth movement animation 丝滑的移动轨迹动画

## How to Run 如何运行
Download the APK from the Releases page and install it, or clone this project and build it in Android Studio.
请在 Releases 页面下载 APK 安装，或克隆本项目到 Android Studio 编译。
